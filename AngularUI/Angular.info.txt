1. Create New Project ng new progect-name 

1.2. Add bootstrap 
npm install bootstrap
Add Bootstrap CSS:In angular.json: 
"styles": [
  "node_modules/bootstrap/dist/css/bootstrap.min.css",
  "src/styles.css"
]

2. Add Following Components 
ng g c pages/home
ng g c pages/register
ng g c pages/login
ng g c pages/Categories 

3. Update app.routes.ts | to  Export routes you wont in navigation, Its a array of path & components. 
	export const routes: Routes = [
    {path: '', component: Home},
    {path: 'login', component: Login},
    {path: 'register', component: Register},
    {path: 'categories', component: Categories}
	]; 
4. app.component.ts | ensure you are  importing : [RouterOutlet,RouterLink], 
5. app.html | Update navigation bar HTML for menu.

<nav class="navbar navbar-expand-lg navbar-light bg-light shadow-sm mb-4">
  <div class="container-fluid">
    <a class="navbar-brand fw-bold" routerLink="/">My App</a> 
    <div class="collapse navbar-collapse">
      <!-- Left-aligned Menu -->
      <ul class="navbar-nav me-auto mb-2 mb-lg-0">
        <li class="nav-item"><a class="nav-link" routerLink="/" routerLinkActive="active-link">Home</a></li>
        <li class="nav-item"><a class="nav-link" routerLink="/categories" routerLinkActive="active-link">Categories</a></li>
      </ul>

      <!-- Right-aligned Menu -->
      <ul class="navbar-nav ms-auto mb-2 mb-lg-0">
        <li class="nav-item"><a class="nav-link" routerLink="/login" routerLinkActive="active-link">Login</a></li>
        <li class="nav-item"><a class="nav-link" routerLink="/register" routerLinkActive="active-link">Register</a></li>
      </ul>
    </div>
  </div>
</nav>

<router-outlet />


6. Update main.ts to provide above routes to main file.

import { routes } from  './app/app.routes';
import { provideRouter } from '@angular/router';

bootstrapApplication(App, {...appConfig,
  providers: [provideRouter(routes)]
}).catch((err) => console.error(err));
